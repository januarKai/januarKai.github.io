Python Conference Indonesia 2021 Lightning Talks

Python Conference Indonesia 2021 (PyConID 2021) menjadi sangat berkesan karena saya juga merupakan salah satu organizer untuk acara tersebut. Tidak seperti bayangan awal saya tentang suatu komunitas bahasa pemrograman, di mana pasti orang-orangnya kaku dan "geeks" banget, ternyata tidak demikian. Teman-teman yang tergabung dari berbagai komunitas python daerah di Indonesia benar-benar menjalankan acara ini dengan menyenangkan namun tetap terorganisir dengan baik. Kembali tentang CUDA Python, yaitu Python API untuk CUDA Programming (mungkin akan saya bahas di blog ini dalam waktu dekat) yang memungkinkan kita melakukan pemrosesan paralel suatu instruksi komputer menggunakan CUDA Core dari GPU NVIDIA. CUDA Python bisa menjadi salah satu framework yang akan sangat membantu teman-teman yang bekerja di bidang image processing, khususnya yang terbiasa menggunakan Python. Sebelum ini, CUDA Programming masih terbatas menggunakan C++ API, sehingga sedikit menyulitkan untuk teman-teman yang belum terbiasa dengan C++. Untuk contoh penggunaan dan script dari CUDA Python bisa dilihat di Lightning Talk saya di PyconID 2021 berikut.
